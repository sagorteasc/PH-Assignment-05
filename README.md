## Q-1: What is the difference between var, let, and const?

**Ans:**

i) **var** - It can be re-declared and update the value of a variable, and it hoists the data to the top.

ii) **let** - It gives the scope to the user to update a variable value when needed, but it cannot be re-declared.

iii) **const** - It is fixed. The user can't update the value of a variable, and it cannot be re-declared.

## Q-2: What is the spread operator (...)?

**Ans:**

The spread operator is used to copy reference type data like arrays and objects. If we want to change something in an array or object, then the original array also changes. So we use the spread operator to copy the array or object and then change or add something to it. By doing this, it does not affect the original array.

---

## Q-3: What is the difference between map(), filter(), and forEach()?

**Ans:**

i) **map()** - It loops through an array, performs functionality on each item, and returns a new array.

ii) **filter()** - It loops through an array and returns the only items that fulfill the condition.

iii) **forEach()** - It loops through an array and performs functionality on each item, but it does not return anything.

## Q-5: What are template literals?

**Ans:**

Template literals (``) are used instead of single quotation or double quotation. They are used for writing multi-line code easily and for dynamic data rendering.
